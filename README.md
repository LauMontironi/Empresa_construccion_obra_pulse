# ObraPulse Frontend

Frontend en Angular para **ObraPulse**, un CRM interno de gestión de obras para empresa constructora.

## Funcionalidades principales

- Landing page pública
- Inicio de sesión con JWT
- Panel de operarios
- Panel de admin
- Gestión de obras
- Notificaciones
- Fotos de obra
- Guards e interceptors para autenticación

## Tecnologías

- Angular
- TypeScript
- CSS
- SweetAlert2

## Estructura base

- `Shared/` componentes reutilizables
- `pages/admin/` vistas del panel admin
- `pages/Operarios/` vistas del panel operario
- `services/` llamadas al backend
- `interfaces/` modelos TypeScript
- `guards/` protección de rutas
- `interceptors/` manejo automático del token
